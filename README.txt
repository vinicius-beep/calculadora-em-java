Calculadora em Java

Este projeto é um código simples de uma Calculadora utilizando a linguagem de programação Java. A calculadora realiza operações básicas como soma, subtração, multiplicação e divisão.

Funcionalidades:

Soma: Adiciona dois números.
Subtração: Subtrai o segundo número do primeiro.
Multiplicação: Multiplica dois números.
Divisão: Divide o primeiro número pelo segundo.

Como usar:

Tenha o Java instalado.
Compile o código.
Execute.
